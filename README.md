# Dict_Scrapper
Launch oxford dict from terminal 

```
# call it like
# python scrapper.py word --no_examples

# there can be separated examples block for one meaning 

# can't handle compex words like take off
# it should be "take+off"

# For some words definition isn't printed
# 'propulsion'
# `trade-off`

# transciprions are badly formatted

# some words have multiple meanings sitting on different links
```

### Example
```bash
oxdict main
Word: MAIN
Phonetics:
Br: /meɪn/
Am: /meɪn/
---------------------
DEFINITIONS:

#1
being the largest or most important of its kind

Examples:
0) We have our main meal at lunchtime.
1) Reception is in the main building.
2) Poor housing and unemployment are the main problems.
3) Loss of habitat is the main reason for the bird's decline.
4) The main thing is to stay calm.
```
